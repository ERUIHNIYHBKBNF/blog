# 建立个人 profile 页面

新建一个仓库跟 github 用户名同名，会看到这样的系统提示：

ERUIHNIYHBKBNF/ERUIHNIYHBKBNF is a special repository.

Its README.md will appear on your public profile.

这个仓库的 README 会被展示到个人主页上，这样就可以愉快地写一个漂亮的个人页面啦 QwQ
