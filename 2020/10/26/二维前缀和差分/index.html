# 题目大意

​ 给出 n×mn \times mn×m 矩阵， 给定 a,ba, ba,b ，在矩阵中任意次选择 a×ba \times ba×b 子矩阵，使其中每个元素均减一， 问是否能减成零矩阵。

T (1≤T≤100)T\ (1\le T \le 100)T (1≤T≤100) 组数据， n,m,a,b (1≤n,m≤1000,1≤a≤n,1≤b≤m)n,m,a,b\ (1\le n,m \le 1000, 1\le a\le n, 1\le b\le m)n,m,a,b (1≤n,m≤1000,1≤a≤n,1≤b≤m)

# 算法描述

​ 对于最左上角的格子， 要把它变成零， 只能选择以它自己为左上角的子矩阵。这样顺着往下推就好啦，，让每个格子都当一次左上角（注意判定距离边界不要小于 a 或 b 什么的。。）看看最后能不能变成零矩阵就好了 |･ω･｀)

​ 剪枝：若中途出现某点为负值， 则直接判定为否 QAQ
